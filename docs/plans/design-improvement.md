# デザイン改善計画

作成日: 2026-04-02
対象: フロントエンド（Angular + Angular Material）

---

## 現状の問題点

### 全体
- Angular Material のデフォルト Indigo テーマそのままで個性がない
- 子ども向け画面と親向け管理画面で視覚的な差別化がない
- 余白・タイポグラフィが最低限で、洗練された印象に乏しい

### 各画面
- **ログイン**: アイコン + ボタンのみのシンプルすぎる構成
- **ChoreRegister（子ども向け）**: カードが白一色で味気ない、選択済み状態のフィードバックが弱い
- **Dashboard**: カードの情報密度が低く残高が目立たない
- **ChildDetail**: 残高表示が埋もれている、収支リストが単調
- **フォーム画面（child-new/edit, record-new等）**: 全ページ同じ構造で個性なし

---

## 改善方針

### 1. カラーテーマの刷新

現在の Angular Material `indigo-pink` テーマを、子ども向けアプリらしい温かみのあるカスタムテーマに変更する。

**採用案: ブルー × イエロー（清潔感 + 活気）**

```
Primary:    #1565C0 (Deep Blue)        → 信頼感・親向け管理
Accent:     #FDD835 (Yellow)           → 明るさ・子ども向けアクセント
Warn:       #E53935 (Red)              → 支出・削除・警告
Income:     #2E7D32 (Green)            → 収入表示（既存の値を継続）
Background: #F3F4F9 (Light Blue Gray)  → 全体背景
```

### 2. タイポグラフィ改善

- Google Fonts から **Noto Sans JP** を導入（日本語の視認性向上）
- 見出しフォントサイズの統一・階層化
- 行間・文字間隔の調整

### 3. グローバルスタイルの整備

`styles.scss` にデザイントークンを定義し、全体の一貫性を確保する。

```scss
// デザイントークン
$spacing-xs: 4px;
$spacing-sm: 8px;
$spacing-md: 16px;
$spacing-lg: 24px;
$spacing-xl: 32px;

$border-radius-sm: 8px;
$border-radius-md: 12px;
$border-radius-lg: 16px;

$shadow-card: 0 2px 8px rgba(0, 0, 0, 0.12);
$shadow-card-hover: 0 4px 16px rgba(0, 0, 0, 0.2);
```

## 画面別改善詳細

### 画面 1: ログイン (`/login`)

**現状**: アイコン + タイトル + ボタンのみ
**改善点**:
- 背景にグラデーションを追加（子どもらしい明るさ）
- アプリアイコンを大きく・目立つ色で表示
- サブタイトル「お手伝いをしておこづかいをもらおう！」を追加
- ログインカードに角丸 + shadow を追加、より立体的に

### 画面 2: ChoreRegister（子ども向けトップ）

**現状**: 白いカードの2列グリッド
**改善点**:
- **子ども選択カード**: 背景色をパステルカラーで子どもごとに色分け
- **種類選択カード**: 金額をカード右上に目立つバッジで表示
- **完了画面**: より大きな達成感（文字・余白・視覚的強調）
- **ヘッダー**: 現在の日付をサブタイトルとして表示

### 画面 3: Dashboard（親向けトップ）

**現状**: シンプルな2列グリッドカード
**改善点**:
- **ヘッダー**: グラデーション背景帯でブランド感を演出
- **子どもカード**:
  - 名前の頭文字をイニシャルアイコン（丸背景）として左側に配置
  - 残高を大きなフォントで強調
  - 残高がマイナスの場合は赤色で警告表示
  - カード左側に 4px のアクセントカラーボーダー
- **子ども追加ボタン**: フローティングアクションボタン（FAB）を右下に配置

### 画面 4: ChildDetail（子ども詳細）

**現状**: 残高カードが浅いインディゴ背景
**改善点**:
- **残高カード**: グラデーション背景（Primary色）+ 白文字で視覚的に際立たせる
- **収支リスト**:
  - 各行に収入（↑ 緑丸）・支出（↓ 赤丸）のアイコンを追加
  - 日付はグレーの小文字で右側に寄せる
- **空データ時**: 「まだ記録がありません」メッセージをセンタリングして表示

### 画面 5: フォーム画面共通（child-new/edit, allowance-type-new/edit, record-new）

**現状**: mat-form-field の羅列
**改善点**:
- フォームカード全体に shadow と角丸を追加して浮き上がり感を出す
- **RecordNew**: 収入/支出トグルをより大きく・カラー強調
  - 収入選択時: 緑のアクセント背景
  - 支出選択時: 赤のアクセント背景

### 画面 6: AllowanceTypeList（種類一覧）

**現状**: mat-list の単純な行
**改善点**:
- カード形式に変更して視認性向上
- 各カードに金額バッジを目立つ形で表示

## 実装時の注意点

- Angular Material のカスタムテーマは Angular 19 の M3 スタイルに対応した形で実装する
- 既存の SCSS クラス名・構造を維持しつつ、スタイルのみ変更することを基本とする
- PWA として動作するため、アニメーションは `prefers-reduced-motion` を考慮する
- 子ども向け画面はタップしやすいサイズ（最小 44px）を維持する
- 色のコントラスト比は WCAG AA 基準（4.5:1）を満たすこと

## デザインコンセプト

**テーマ**: 「おこづかい帳 × 子どもの成長ノート」
- 清潔感のある白 + 明るいブルーで信頼感・安心感を演出
- イエロー/ゴールドのアクセントでお金・コインのイメージ
- 丸みのある角丸・大きめのタッチターゲットで子どもにも使いやすい UI
- アニメーションは控えめに、必要なフィードバックのみ
